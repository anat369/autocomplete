Порядок работы скрипта:

1. Закидываем файлы на сервер (например, локальный - openserver).
2. Запускаем файл index.php, заполняем форму для подключения к базе данных и созданию таблицы с данными (данные берутся из файла test.csv).
3. Параметры подключения к базе сохраняются в файле config.txt
4. После автоматического перехода на страницу выбора товара пробуем что-нибудь найти, сработает автозаполнение.

При заполнении формы неправильными данными будут выскакивать ошибки (например, неверно ввели адрес сервера), это  момент еще не отработал.
